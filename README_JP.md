## About

### Mission

** 3DCGを文字、画像、動画に次ぐWeb上のアセットとして確立する。**

これが我々の定義する唯一無二のjThreeの使命である。

### Our methods

* すべてのエンジニアが自分の本当にやりたい部分に注力できるようにするライブラリ
* Webにおける3Dの役割の再定義
* ContributorのUXを最大限考慮する
* ゲームエンジンではなく、**Web3Dサービスエンジン** である。

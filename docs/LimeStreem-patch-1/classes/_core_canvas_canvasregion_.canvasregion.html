Class CanvasRegion

Abstract class to provide mouse tracking feature on a part of region on canvas. This class is intended to be used in Canvas and viewport renderer.

キャンバス内の特定領域におけるマウスイベントを管理するためのクラス。 主にキャンバス自身や、ビューポートを持つレンダラによる使用を想定されている。
